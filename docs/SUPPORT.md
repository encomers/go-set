# Support

## Getting Help

If you need help using **go-set**, there are several ways to get support.

---

## Questions

For general questions:

* Open a GitHub Issue
* Clearly describe your problem
* Provide code examples if possible

---

## Bug Reports

If you believe you found a bug:

1. Check existing issues
2. Provide a minimal reproducible example
3. Include:

   * Go version
   * OS / architecture
   * Expected vs actual behavior

---

## Feature Requests

Suggestions are welcome.

Please include:

* Use case
* Why the feature is needed
* Possible API design (optional)

---

## Performance Issues

Since this library focuses on performance:

* Include benchmark results (`go test -bench`)
* Describe your workload
* Compare before/after if possible

---

## Important Notes

* This project has **no external dependencies**
* API stability is important, breaking changes are minimized
* Keep discussions focused and technical

---

## Response Expectations

Responses depend on availability of the maintainer.
There is no guaranteed response time.

---

Thank you for using go-set.
